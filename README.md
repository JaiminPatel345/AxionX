# AxionX
Trading  platform using Spring Boot and React
             
### How to start 

**Requirements**
- Node >=20
- Java >=21
- MySQL >=7.0 ( Database with name axionx)


#### Frontend 
``` Bash
     cd Forntend/
     yarn
     yarn dev   
```

#### Backend
``` Bash
     cd Backend/
     ./mvnw clean install 
     ./mvnw spring-boot:run
```
